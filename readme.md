# VK Currency Bot

Бот для конвертации валют во ВКонтакте.

## Возможности

* Перевод валют:

  * евро
  * доллар
  * рубль

## Команды

* /start — инструкция
* /help — инструкция
* /values — список валют

## Пример использования

евро доллар 100

## Установка

```bash
pip install -r requirements.txt
```

Создайте файл `config.py` и вставьте токен:

```python
TOKEN = "ВАШ_ТОКЕН"
```

## Запуск

```bash
python main.py
```
